# Programmatic: Technology Stack & Ecosystem

Purpose: Defines software ecosystem, middleware logic, and partner incentives.



## 1. The Integration Layer (Middleware Logic)
To solve "Messy Reality" data silos, Programmatic implements custom middleware logic to bridge fragmented stacks:

* **NMS to CRM Synchronization**: Real-time webhook listeners (e.g., Splynx/Sonar to HubSpot/Zendesk) ensuring "Source of Truth" alignment for subscriber data.
* **Automated Work Order Triggers**: Logic-based conversion of NMS hardware alerts (down nodes/signal degradation) into prioritized dispatch tickets, eliminating manual triage.
* **Data Normalization & Cleaning**: Python-based scripts designed to format legacy billing data and "veteran-memory" logs into clean datasets for automation engines.



## 2. Supported ISP Technology Stack
Programmatic is tool-agnostic but specializes in the following industry-standard nodes:

### 1. Network & Provisioning (NMS)
- **Primary**: Splynx, Sonar, Powercode, UISP (Ubiquiti), MikroTik (WinBox/The Dude).
- **Enterprise/Carrier**: Nokia Altiplano, Huawei iMaster NCE, Juniper Mist.
- **Monitoring**: Zabbix, LibreNMS, PRTG.

### 2. Billing, CRM & Support
- **BSS/Billing**: Sonar, Splynx, Stripe Billing, Chargebee, GoCardless.
- **CRM & Ticketing**: Salesforce Service Cloud, HubSpot, Zendesk, Front.

### 3. Field Service & Ops (FSM)
- **Dispatch**: Jobber, ServiceTitan, Skedulo.
- **GIS/Mapping**: VETRO FiberMap, ArcGIS, Google Maps API.

### 4. Automation & AI Layer (The "Glue")
- **Workflow Engines**: Make.com, n8n (Self-hosted), AWS Lambda.
- **AI Processing**: Gemini (Log analysis/SOP generation), Claude (Reasoning/Logic), Fireflies (Transcription).
- **Custom Development**: Python (FastAPI), Node.js serverless bridges.

## 3. Operator Referral & Partner Program
**Objective**: Transform operational excellence into a quiet, compounding revenue stream to offset build costs.

* **Eligibility**: Vetted ISP operators only.
* **Reward Structure**: Recurring monthly commissions for every qualified introduction that signs a build contract.
* **Technical Tracking**: 
    - Dedicated partner landing page and unique tracking links.
    - Automated text/email notifications for pipeline stage changes (from Audit to Signed).
    - Transparent dashboard for monitoring partner income.
